# Python学生信息管理系统 python78

## 项目概述

本项目是一款基于Python的学生信息管理系统，集成了学生管理、成绩管理、班级管理、课程管理、宿舍管理、任务管理和资料管理等模块，为学校提供一个高效的信息管理解决方案。

## 技术栈

- **前端**：HTML、CSS、JavaScript、jQuery
- **后端**：Python、Flask
- **数据库**：MySQL
- **开发工具**：VSCode、Navicat
- **运行环境**：Python 3.0以上、MySQL 8.0

## 功能模块

- 登录
- 注册
- 学生管理
- 成绩管理
- 班级管理
- 课程管理
- 宿舍管理
- 任务管理
- 资料管理
- 意见反馈
- 系统管理

## 系统角色

系统主要分为以下角色：
- 学生
- 教师
- 管理员

各角色拥有不同权限，以满足不同用户的需求。

## 运行环境

- Python 3.0及以上版本
- MySQL 8.0及以上版本

## 部署步骤

1. 安装Python 3.0以上版本
2. 安装MySQL 8.0以上版本
3. 安装项目依赖库（Flask等）
4. 创建数据库，导入数据表结构
5. 修改配置文件，连接数据库
6. 运行项目
